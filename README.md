# GUI-Python-Calculator

iPhone Style Calculator
A simple iPhone-style Calculator built using Python Tkinter.

Features
•	Modern iPhone-style interface
•	Addition (+)
•	Subtraction (-)
•	Multiplication (*)
•	Division (/)
•	Clear (C)
•	Equals (=)

•	Error handling
•	Responsive layout
Technologies Used
•	Python 3
•	Tkinter

Project Structure
•	Calculator/
•	├── calculator.py
•	└── README.md

How to Run
•	1. Install Python from https://www.python.org/downloads/
•	2. Open terminal in project folder.
•	3. Run: python calculator.py

Application Preview
•	Number buttons (0–9)
•	Arithmetic operators
•	Clear button
•	Result display
•	Black iPhone-inspired theme

Functions
•	Numbers: Input digits
•	+ : Addition
•	- : Subtraction
•	* : Multiplication
•	/ : Division
•	C : Clear screen
•	= : Calculate result

How It Works
•	Click buttons to build expression.
•	Press = to evaluate.
•	Result is displayed.
•	Invalid expressions show an error dialog.

Requirements
•	Python 3.x
•	Tkinter (included with Python)
